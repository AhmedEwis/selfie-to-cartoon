## Installation Options
1. Make sure you have python3 installed:
    + `$ python --version` \
    or
    + `$ python3 --version`
2. Install with [`pip`](https://pypi.org/project/stronghold/) 
    + `$ pip install -r requirements.txt` 

## Usage
To run locally in command line run:
  ```sh
$ streamlit run app.py
```
